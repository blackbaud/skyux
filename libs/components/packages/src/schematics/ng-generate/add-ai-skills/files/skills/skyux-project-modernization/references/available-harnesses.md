# Available SKY UX Test Harnesses

**Load this reference when:** writing tests for SKY UX components, migrating from `Sky*Fixture` to `Sky*Harness`, or looking up which harness exists for a component.

This list is verified by a unit test that scans the monorepo. If a harness is added or removed, the test will fail with the exact list of changes needed.
