Using the bbHelpOpenOnClick directive

  • Grab By Keyboard
  • anchor link: default.html
  • button:

Setup:

  • Has a bb-help-key through local demo component toggle-key switching between two pageDefaultKey's: example1.html and gonxt-workflow.html.html.

  • Has a bb-help-button with the helpKey default.html.

  • Has a Sky Modal with a bb-help-key with the helpKey example3.html.

Expected Behavior:

  • Invoker/Toggle Widget: Should load the current toggled pageDefaultKey, either example1.html or gonxt-workflow.html.

  • Help Button: Should open the default.html help page from bb-help.

  • Invoker/Toggle Widget: (with modal open) Should open the example3.html help page from bb-help.
